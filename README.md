# Password Manager

This is my first training project - a password manager written in Java, at the moment, **it is a raw version without encryption**, an ordinary skeleton of a full-fledged program in the future.
It can store password records from services, but does not encrypt them, **the program also dies after completion**.

## Functionality

- Authorization
- creating records
- viewing records
- editing records

## Plans

In the future, I plan to refine this program as I improve my programming skills.
I plan to add data persistency, for this I think to connect SQL, also necessarily in the plans - this will add password encryption for records and hashing for the master password. 
In the future, it is planned to make it a full-fledged non-console application with a graphical interface

## Stack

Java
